Installation and DependenciesÂ¶

Installing PySurfer is quite simple with easy_install:

easy_install -U pysurfer

or pip:

pip install pysurfer

To use PySurfer, you will need to have the following Python packages:

  • numpy
  • scipy
  • ipython
  • nibabel
  • mayavi
  • matplotlib
  • PIL

An easy option to get all of these packages is to use the Enthought Python Distribution (EPD) which is free for academic use.
