===================
Save a set of views
===================

Save some views in png files.




.. code-block:: python

    from surfer import Brain

    print(__doc__)

    sub = 'fsaverage'
    hemi = 'lh'
    surf = 'inflated'

    brain = Brain(sub, hemi, surf)


save 1 image



.. code-block:: python

    brain.show_view('lat')
    brain.save_image("%s_lat.png" % sub)


save some images



.. code-block:: python

    brain.save_imageset(sub, ['med', 'lat', 'ros', 'caud'], 'jpg')
